**Rock Paper Scissors Game **(JavaScript)

A simple and interactive Rock Paper Scissors game built using HTML, CSS, and JavaScript.
The game allows users to play against the computer, track scores using localStorage, and even enable an auto-play mode.

**Features**:
  Play Rock, Paper, Scissors against the computer
  
  Persistent score tracking using localStorage
  
  Auto Play mode (computer plays every second)

Keyboard shortcuts:
  R → Rock
  P → Paper
  S → Scissors

Reset score functionality

Clean and responsive UI

Technologies Used

  HTML5 – Structure of the game
  
  CSS3 – Styling and layout
  
  JavaScript – Game logic and interactivity

**Todo List App** (JavaScript)

A minimal Todo List application built using HTML, CSS, and Vanilla JavaScript.
This project helps users add tasks with due dates and delete them dynamically using DOM manipulation.

**Features**

  Add a todo with a name and due date

  Delete todos instantly
  
  Grid-based layout using CSS Grid
  
  Dynamic rendering with JavaScript
  
  Beginner-friendly and easy to understand code

**Technologies Used**

HTML5 – Page structure

CSS3 – Styling and grid layout

JavaScript – Logic, events, and DOM manipulation
